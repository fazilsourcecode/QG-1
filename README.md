# Quick-Grade
📄 Quick Grade: Automated Mark Sheet Analyzer Quick Grade lets users upload scanned or photographed marksheets, automatically extracts handwritten marks per question, maps them to student RRNs, calculates totals, and exports everything into a structured Excel sheet — all with ease.
🎯 Built to save time, reduce manual effort, and improve accuracy in student assessment processing.

🔍 Key Features
📸 Upload Marksheet Images: Supports scanned or mobile-captured images.

✍️ Handwritten Mark Recognition: AI detects question sections and extracts marks written in adjacent boxes.

🔁 Automated Backend Processing: Uses powerful APIs to handle image analysis and mark recognition.

📊 Excel Sheet Export: Generates organized Excel reports with per-question marks, RRN mapping, and total scores.

⚡ Fast & Intuitive UI: Developed with Next.js and TypeScript for a clean, responsive user interface.

💡 Ideal For
Teachers and examiners who want to digitize paper-based marksheets.

Educational institutions seeking fast and error-free mark processing.

Scenarios requiring bulk marksheet scanning and reporting.

🛠️ Tech Stack
Frontend: Next.js + TypeScript

Backend: Python-based APIs for OCR and AI processing

OCR/AI Models: Google Vision API or custom-trained deep learning model

Excel Export: Using openpyxl or xlsxwriter
